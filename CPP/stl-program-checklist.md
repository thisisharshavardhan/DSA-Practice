
# STL Program Checklist

## 🟢 Level 1 — Basics (Containers & Loops)

- **Vector Input/Output**
	- Take `n` numbers → store in vector
	- Print them

- **Sum of Elements**
	- Use `accumulate()`

- **Find Maximum / Minimum**
	- Use `max_element()` / `min_element()`

- **Count Occurrences**
	- Count how many times `x` appears (`count()`)

- **Reverse a Vector**
	- Use `reverse()`

---

## 🟡 Level 2 — Sorting & Searching

- **Sort Array**
	- Ascending & Descending (custom comparator)

- **Binary Search**
	- Use `binary_search()`

- **Lower Bound / Upper Bound**
	- Find first `≥ x` and first `> x`

- **Remove Duplicates (Sorted Vector)**
	- Use `unique()` + `erase()`

---

## 🟠 Level 3 — Set & Map

- **Unique Elements using set**
	- Input numbers → print unique sorted

- **Frequency Counter using map**
	- Count frequency of each number

- **First Non-Repeating Element**
	- Use `map` or `unordered_map`

- **Check Two Arrays are Same**
	- Use maps or sorting

---

## 🔵 Level 4 — Stack & Queue

- **Balanced Parentheses**
	- Use `stack`

- **Next Greater Element**
	- Classic stack problem

- **Queue Simulation**
	- Push, pop, front

- **Reverse Queue**
	- Using stack

---

## 🟣 Level 5 — Deque & Priority Queue

- **Sliding Window Maximum (basic version)**
	- Use `deque`

- **K Largest Elements**
	- Use `priority_queue`

- **Min Heap / Max Heap Practice**
	- Insert + extract

---

## 🔴 Level 6 — Algorithms Practice

- **Check Palindrome (using STL)**
- **Merge Two Sorted Vectors**
- **Find Intersection of Two Arrays**
- **Rotate Array**
	- Use `rotate()`
- **Permutation of String**
	- Use `next_permutation()`
